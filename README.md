# Darjeeling Homestay Booking Platform

A complete homestay booking platform for Darjeeling and nearby tourist destinations built with Django and PostgreSQL.

## Features

### Customer
- Register and Login
- Search Homestays
- Filter by Price and Location
- View Homestay Details
- Book Rooms Online
- Add to Wishlist
- Submit Reviews and Ratings

### Homestay Owner
- Register as Owner
- Create and Manage Homestays
- Upload Homestay Photos
- Manage Rooms and Pricing
- View Bookings
- Update Availability

### Admin
- Approve Homestays
- Manage Users
- Monitor Bookings
- View Platform Statistics
- Manage Reviews

## Technology Stack

- Python 3
- Django 5
- PostgreSQL
- Bootstrap 5
- HTML, CSS, JavaScript
- GitHub
